# CareBridge

A clinical decision support (CDS) mini-service that identifies patients overdue for screenings (e.g., colonoscopies) and provides a clinician dashboard for initiating automated notifications.

> [!CAUTION]
> **Active Development:** This project is evolving rapidly. Expect breaking changes and frequent updates.

### **Tech Stack**

- **Backend:** ASP.NET Core 10 Web API (RESTful).
- **Frontend:** Angular 21.
